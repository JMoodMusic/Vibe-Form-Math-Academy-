# 수학 학원 상담/수강 문의 예약 서비스 - 요구사항 명세서 (MVP)

## 1. 서비스 한 줄 정의
초·중·고 학생/학부모가 수학 학원 상담 및 레벨 테스트 예약을 온라인으로 신청하고, 관리자가 이를 확인·배정·관리하는 서비스.

## 2. 사용자 종류

### 학부모/학생 (고객)
- 상담 신청
- 레벨 테스트 예약
- 시간 선택

### 관리자 (원장/상담 담당자)
- 신청 내역 조회
- 학생 정보 확인
- 상담 일정 확정
- 상태 관리
- 상담 결과 기록

## 3. 핵심 기능 (MVP)

### 고객 기능
1. 상담/레벨테스트 유형 선택
   - 상담 신청
   - 무료 레벨테스트 예약
2. 학생 정보 입력
   - 이름, 학년, 학교, 보호자 연락처, 현재 수학 성적 수준
3. 희망 일정 선택
   - 날짜 선택
   - 시간대 선택 (14:00~21:00, 1시간 단위)
4. 제출 완료 메시지
   - "담당자가 확인 후 연락드립니다."

### 관리자 기능
1. 신청 목록 조회 - 필터 (학년 / 상태 / 날짜)
2. 상세 정보 확인
3. 상태 변경 - 접수됨 → 연락완료 → 상담확정 → 완료 → 취소
4. 상담 메모 작성 - 현재 수준, 추천 반, 상담 내용 요약

## 4. 화면 목록

| 경로 | 설명 |
|------|------|
| `/` | 학원 소개 + 상담 예약 버튼 |
| `/reserve` | 상담/레벨테스트 신청 폼 |
| `/complete` | 신청 완료 페이지 |
| `/admin` | 관리자 목록 페이지 |
| `/admin/:id` | 개별 신청 상세 |

## 5. 데이터 항목 설계

### 기본 정보
| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| `student_name` | 필수 | 학생 이름 |
| `grade` | 필수 | 학년 (초1~고3) |
| `school` | 선택 | 학교명 |
| `parent_phone` | 필수 | 보호자 연락처 |
| `parent_name` | 선택 | 보호자 이름 |

### 학습 정보
| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| `current_math_level` | 선택 | 상 / 중 / 하 |
| `recent_exam_score` | 선택 | 최근 모의고사 점수 |
| `exam_target` | 선택 | 내신 / 수능 / 특목고 대비 |

### 예약 정보
| 필드 | 필수 여부 | 설명 |
|------|-----------|------|
| `reservation_type` | 필수 | 상담 / 레벨테스트 |
| `desired_date` | 필수 | 희망 날짜 |
| `desired_time_slot` | 필수 | 희망 시간대 |

### 관리 정보
| 필드 | 설명 |
|------|------|
| `status` | 접수 / 연락완료 / 확정 / 완료 / 취소 |
| `admin_memo` | 관리자 메모 |
| `created_at` | 신청 일시 |
| `updated_at` | 최종 수정 일시 |

## 6. MVP 제외 항목
- 회원가입 / 로그인
- SMS 자동 발송
- 예약 시간 중복 자동 차단
- 결제
- 반 자동 배정 로직
- 출결 / 성적 관리 기능
